>[!quote] Описание
_Когда магическое число без переменной возникает в коде, для читателя это всегда неожиданность, ведь абсолютно не понятно, что оно означает._

***
### Пример

> [!fail]
>```python
>def calculate(value):
>	return value + 108
>```

> [!success]
> ```python
>def calculate(value):
>	magic_num = 108
>	return value + magic_num
>```
