>[!quote] Описание
_При изменении количества в значении переменной придётся менять и название самой переменной, поэтому числа в названиях не используются._

***
### Пример

> [!fail]
> ```python
> five_dogs = 5
> ```

> [!success]
> ```python
> dogs_num = ...
> ```

> [!success]
> ```python
> dogs_count = ...
> ```
