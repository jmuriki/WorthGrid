>[!quote] Описание
_Ключевое слово должно располагаться в конце названия.
Неправильный порядок слов усложняет чтение._

> [!fail]
> ```python
> num_active_users = ...
> ```

> [!success]
> ```python
> active_users_num = ...
> ```

### Пример 3

> [!fail]

> 
>     def convert_amount(self, amount):
