### prc? qnt?
- [ ] Пример: prc? qnt?

```python
"""Плохо"""
class Product:
    def __init__(self, name, prc, qnt):
        self.name = name
        self.prc = prc
        self.qnt = qnt


"""Хорошо"""
class Product:
    def __init__(self, name, price, quantity):
        self.name = name
        self.price = price
        self.quantity = quantity
```
